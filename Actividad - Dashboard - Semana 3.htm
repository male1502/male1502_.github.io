Descripcin paso a paso modelado de datos:

 

1.       Importar la base de datos:

Abre Looker Studio y accede al rea de administracin de datos o al lugar donde puedes importar nuevas fuentes de datos.

Utiliza la URL proporcionada: (https://docs.google.com/spreadsheets/d/1IPS5dBSGtwYVbjsfbaMCYIWnOuRmJcbequohNxCyGVw/edit?resourcekey#gid=1625408792) para importar la base de datos "Ask A Manager Salary Survey 2021"

2.     Limpiar los campos de "Country" y "City":

Una vez que la base de datos est importada, puedes utilizar herramientas de limpieza de datos dentro de Looker Studio para homogeneizar los nombres de los pases y ciudades. Esto puede implicar la eliminacin de espacios en blanco adicionales, correccin de errores tipogrficos, estandarizacin de nombres, etc.
Para esto, debes ingresar a la pestaa Recursos > Gestionar las fuentes de datos aadidas > Editar, hacer el cambio del tipo de variable a las que corresponde y guardar los cambios.

3.     Crear los campos nuevos "salario anual" y "compensaciones" en pesos colombianos:

Necesitars acceder a la configuracin de campos dentro de Looker Studio, de la misma manera que en el punto (2), con la adicin del paso Recursos > Gestionar las fuentes de datos aadidas > Editar > aadir un campo.

Una vez aadido el campo, designa el nombre de la nueva variable e introduce la formula o el valor especifico de la columna. En este caso, plasmaramos el siguiente cdigo:

WHEN Moneda = "USD" THEN Otras compensaciones * 3956.54 y as sucesivamente para cada una de las monedas existentes dentro de la Data

Reminder: Utiliza la tasa de cambio del da en que realices el ejercicio para convertir los sueldos y compensaciones a pesos colombianos.

Para las compensaciones, simplemente repite los pasos anteriores.

4.     Crear un campo adicional sumando salario anual y compensaciones en pesos colombianos:

De manera similar, puedes utilizar la herramienta de configuracin de campos en Looker Studio (repite los pasos del punto 3) para crear un nuevo campo que sume el salario anual y las compensaciones ya convertidas a pesos colombianos, la formula seria:

Salarios (pesos colombianos) +Compensaciones (pesos colombianos)
